v0.2 — Tunnel UX polish

2026-04-30
  • featNew tunnel ssh button per row that copies sh <(curl -sS .../tunnel-listen.sh) CODE to the clipboard for pasting in a host terminal. Tooltip explains the host-paste workflow.
